关于一些IDE的推荐

信息技术 IDE(编辑器)
Publish Date:   2024-06-08
Update Date:   2024-06-08
Word Count:   815
Read Times:   2 Min
Read Count:  

前言

为了给我们学校的信息技术老师建议使用一些其他的 IDE 而不要再用 IDLE（Python自带编辑器），这篇文章出现了。

附：这里不只有 Python 编辑器推荐，其他的单语言编辑器和多语言编辑器在这里都会有出现（不考虑vim、nano类编辑器和各语言内置编辑器（如果有的话，如 IDLE，古早编辑器如 UltraEdit 也不会提及，当然绝对不是因为我太菜了不知道），如果你是 OIer，也可以往下看。

多语言编辑器

省流表

Sublime Text Visual Studio Code
资源占用 较低 较高
连字/git/ARM64平台支持 支持 支持
调试 不支持 支持
WSL/Docker/SSH 等容器/远程开发支持 非官方支持 官方支持（插件）
Code Coverage/Profiling等功能 非官方支持 非官方支持
Linux 软件包管理器支持（仅考虑apt/dnf(yum)/pacman） 第三方支持+手动配置 官方支持(VSCodium)+第三方支持+手动配置
收费 是（试用期不影响使用） 否

Sublime Text

下载链接：https://www.sublimetext.com/

点击上方链接，点击首页 Download 即可。如果下载速度较慢可用魔法或去国内流氓软件站下载。

欢迎 xhgua 发 Pull Request 来给这里添加内容，我是真没用过（逃）

Visual Studio Code（VSCode）

下载

下载链接：code.visualstudio.com

选择中间大大的下载按钮即可。

如果你没有梯子或者下载速度太慢，可以通过国内流氓软件站下载，也可以参照如下方法：

将下载中的VSCode的链接复制下来，再在浏览器新开一个标签页，将刚刚复制好的链接粘贴进去。然后，将链接中的 https://az764295.vo.msecnd.net 替换成 https://vscode.cdn.azure.cn 即可。这一步是将下载地址改为国内的镜像地址。

注意不要下到 CEC-IDE。

打开直接按步骤安装即可。

配置

打开 VSCode ，不管新手教程，直接点击右侧的四个方块的图标（Extension）,

图中最下面这个按钮

在上面的搜索框中输入 Chinese ，选择这个拓展：

点击如图所示“Install”按钮：

重启软件，恭喜你获得了一个中文的编辑器（逃）

然后在拓展里搜索你想要的语言拓展安装即可。

如：我想要安装Python，那么我需要先搜索 Python，找到由 Microsoft 官方提供的 Python 拓展（包），然后安装。

注：VSCode 可以直接打开工作区/文件夹。

推荐拓展：Black Formatter(Microsoft)(Python)，One Dark Pro(binaryify)(界面主题)

Python 编辑器

注：部分编辑器可能同时支持其他少量语言或其他文件的查看，如 Pycharm Professional 支持查看 SQLAlchemy 和 SQLite 的数据库文件和直接修改。

PyCharm Spyder
收费 部分版本收费（专业版） 免费
资源占用 较高 较低/中等
调试 支持 支持
WSL/Docker/SSH 等容器/远程开发支持 非官方支持 不支持
Code Coverage/Profiling/git/连字等功能 原生支持 不支持
Linux 软件包管理器支持（仅考虑apt/dnf(yum)/pacman） 第三方支持+手动配置 官方支持
